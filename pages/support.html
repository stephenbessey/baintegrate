FAQ

Frequently Asked Questions

Quick answers to the most common questions about BAIS integration

Getting Started

How do I get started with BAIS integration?

Start by creating an account and obtaining your API key. Then follow our quick start guide to make your first API call and set up basic integration.

What business types does BAIS support?

BAIS currently supports hotels, restaurants, and retail businesses. Each business type has specific operations like search, book, modify, and cancel.

Do I need programming experience to use BAIS?

While some programming knowledge is helpful, we provide comprehensive documentation, SDKs, and integration guides to help users of all technical levels.

Technical Support

What if I encounter API errors?

Check our troubleshooting guide first. If you continue to have issues, contact our support team with the specific error message and your request details.

How do I handle rate limiting?

Implement exponential backoff and respect the rate limits specified in our API documentation. Consider upgrading your plan if you need higher limits.

Can I test integrations before going live?

Yes, we provide sandbox environments and test endpoints for all our integrations. This allows you to test thoroughly before deploying to production.

Billing & Plans

How does billing work?

We offer monthly and annual subscription plans with different usage limits. Usage-based fees apply for API calls beyond your plan limits.

Can I change my plan anytime?

Yes, you can upgrade or downgrade your plan at any time. Changes take effect immediately, and we'll prorate any billing differences.

Is there a free trial available?

We offer a 14-day free trial for all plans with full access to our platform and support. No credit card required to start.
